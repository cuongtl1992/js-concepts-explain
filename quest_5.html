Event Propagation Visualization

The Problem:

Given the following HTML and JavaScript:

<div id="outer">
    <button id="inner">Click me</button>
</div>

document.getElementById('outer').addEventListener('click', () => {
    console.log('Outer div clicked');
});

document.getElementById('inner').addEventListener('click', (e) => {
    e.preventDefault();
    console.log('Inner button clicked');
});

If a user clicks the button, what will be logged to the console?

Answer: Both "Inner button clicked" and "Outer div clicked" will be logged because preventDefault() only prevents the default action of the element (like form submission) but doesn't stop event propagation. For that, you'd need stopPropagation().

Case 2: stopPropagation()
